# Pixiv Tag Collector

Pixiv の一覧ページから少数件の作品ページを開き、タグ情報を JSON としてエクスポートする Chrome 拡張です。

## 初回の前提

- 取得は明示実行のみ
- 上限は `5 / 10 / 20` 件
- 1件ずつ順番に処理
- 既存の seed / learning へ自動反映しない
- 出力は JSON ダウンロードまで

## 使い方

1. Chrome の拡張機能画面で `デベロッパーモード` をオンにする
2. `パッケージ化されていない拡張機能を読み込む` でこのフォルダを指定する
3. Pixiv の一覧ページを開く
4. 拡張 popup で `source_key`、件数、待機時間を指定する
5. `一覧から作品タグを収集` を押す
6. JSON ダウンロードを確認する

## 出力

JSON には次が入ります。

- `source_key`
- `list_page_url`
- `requested_count`
- `success_count`
- `failed_count`
- `items[]`

`items[]` ごとに、作品 URL、作品タイトル、タグ一覧、取得成否を持ちます。

## 注意

- Pixiv 側 DOM 変更で取得できなくなる場合があります
- 取得対象を広げすぎないでください
- 作品ページでタグが見つからなかったものは `failed` として出力されます
