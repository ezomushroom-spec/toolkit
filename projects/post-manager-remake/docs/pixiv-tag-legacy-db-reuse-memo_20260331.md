# 旧 pixiv-post-manager タグDB 再利用メモ 2026-03-31

## 現状確認

- 参照元は [pixiv_manager.db](/E:/自作アプリ集/ボツ/pixiv-post-manager/data/pixiv_manager.db)。
- `tags` テーブルは 145 件ある。
- `artworks` は 1 件、`artwork_tags` は 0 件で、作品実績の集積というより「過去に溜めたタグ集積」に近い。
- `category` は空欄が 101 件で、構造化は弱い。
- 一方で `usage_count` は一部タグに入っており、R18寄り語彙の優先度判断には使える。

## 採用判断

### 1. このDBは `seed` の正本ではなく「語彙棚卸し原料」として使う

- 今の [pixiv_seed_tags.json](/E:/codex/workspace/projects/post-manager-remake/app/data/pixiv_seed_tags.json) へ丸ごと流し込む用途には向かない。
- 理由は、作品タグ、キャラタグ、R18語彙、衣装、ノイズ、古い作品依存語彙が未整理で混在しているため。
- ただし `usage_count` があるので、「何がよく使われていたか」を見る材料としては有用。

### 2. 再利用の向き先は2つある

- `seed候補の拾い出し`
- `learning/history 側へ寄せる語彙候補の抽出`

### 3. いま有用度が高いのは R18寄りの内容語彙

- 上位には `R-18`, `おへそ`, `むちむち`, `おっぱい`, `すじ`, `はいてない`, `くぱぁ`, `アヘ顔`, `中出し`, `ロリ` が入っていた。
- 0件でも、`オナニー`, `イチャラブ`, `パイズリ`, `フェラ`, `バニーガール`, `メイド`, `巨乳`, `騎乗位` など、今の支援で使いたい語彙がある。
- 逆に `AI`, `AI生成`, `girl` などは、そのまま現在の用途へ持ち込むか慎重に見るべき。
