# App Rules

- この `app/` を正本実装先として扱う。
- 旧配置 `E:\自作アプリ集\narrative-thumbnail-studio` は比較用に残っているが、今後の主変更先はここに固定する。
- `preset_panel_preview*.png`、`test_*.png`、`startup_error.log`、`_pythonw_*` は生成物や確認用ファイルとして扱い、明示依頼なしで編集しない。
- `presets/*.json` はプリセット定義の正本候補なので、UI 側の変更と混ぜて無断で変えない。
- 画像読み込み、出力、履歴クリア、プロジェクト保存/読込は危険操作寄りなので、確認手順を残して変更する。
