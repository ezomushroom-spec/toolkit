# 次段階準備

## 現在の到達点

- 起動バッチでアプリを起動できる状態
- レイアウト候補は実運用向けに整理済み
- 画像配置は D&D ベースに整理済み
- 割当順は `MainWindow` 主導に寄せてある
- クロップ保持は `image path` 基準へ移行済み

## 次に進める対象

1. `画像配置` UI の完成度を上げる
   - 左の位置欄と右の D&D リストの高さ・余白調整
   - 未使用画像と使用画像の見え方を分ける
   - 役割名の短縮と統一

2. プリセット切替時の意図保持を詰める
   - `center_hero` と `grid_2x2` の往復
   - `vsplit` 選択時に上位 2 枚を明示
   - おすすめプリセット表示の自然さ確認

3. 操作結果の見え方を安定させる
   - 並び替え直後のプレビュー反映確認
   - パン操作後の画像維持確認
   - プレビューエラー時に古い表示が残り続けない導線づくり

4. 保存/読込の回帰確認
   - `assigned_image_paths`
   - `crop_offsets_by_path`
   - テキスト設定
   - 出力サイズ
