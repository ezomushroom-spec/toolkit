# 手動チェック手順 2026-04-05

## 目的

- 今回修正した高優先不具合を、実 GUI 操作で再確認するための最小手順を固定する

## 事前準備

- `PYTHONPATH=E:\` を通した状態で起動する
- アプリデータは `%LOCALAPPDATA%/CivitaiDownloader/`
- 実運用データを触るため、必要なら事前に DB を退避する

## 起動確認

1. `python -m civitai_downloader` で起動する
2. ブラウズ、ダウンロード、通知、設定の 4 タブが表示されることを確認する
3. 起動直後に即終了しないことを確認する

## DB 保持確認

1. ダウンロードキューに 1 件追加する
2. アプリを閉じる
3. 再起動する
4. 追加したジョブがダウンロード一覧に残っていることを確認する

## 未解決ジョブ失敗確認

1. 解決に失敗する URL または無効なモデルページ相当を使ってジョブを作る
2. 状態が `失敗` になり、ツールチップまたは状態表示から失敗理由を確認できることを確認する
3. 右クリックの `リトライ` で再解決へ戻ることを確認する

## 行マッピング確認

1. ジョブを 3 件以上追加する
2. 中央のジョブを削除する
3. 残った各行で右クリックし、開くページや対象ジョブがずれていないことを確認する

## 危険操作確認

1. ジョブを右クリックして `削除` を押す
2. 確認ダイアログが出ることを確認する
3. `No` で一覧から消えないことを確認する
4. `Yes` で削除されることを確認する

## API 接続テスト確認

1. 設定タブで `接続テスト` を押す
2. テスト中にウィンドウが完全に固まらないことを確認する
3. 成功時または失敗時にダイアログが表示されることを確認する

## 記録すべきもの

- 成功した手順
- 失敗した手順
- 表示されたエラーメッセージ
- 再現に使った URL やジョブ条件
