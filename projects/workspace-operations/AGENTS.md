# workspace-operations

この project は、workspace 自体の運用、棚卸し、計測、改善メモを扱う。
アプリ実装案件ではなく、運用観測と改善判断を残すための project として使う。

## この project で扱うもの

- 計測ルール
- 月次または区切り単位の運用スナップショット
- skill 棚卸しの補足メモ
- workspace 改善の判断メモ

## この project で扱わないもの

- 個別アプリの build / run / test 手順
- 個別案件の仕様書や実装計画
- workspace 共通ルール本体の代替

## 正本

- workspace 共通ルール: `E:\codex\workspace\AGENTS.md`
- 作業フロー: `E:\codex\workspace\docs\workflow.md`
- 品質基準: `E:\codex\workspace\docs\quality-gates.md`
- docs 更新判断: `E:\codex\workspace\docs\documentation-update-rules.md`
- 計測の前段整理: `E:\codex\workspace\docs\effectiveness-measurement-prep.md`

## 変更時の基本

- この project の記録は、共通ルール本体ではなく運用観測として残す
- 数字だけで結論を出さず、未確認事項、残るリスク、詰まりの型も併記する
- 計測の負荷が上がりすぎる場合は、自動化より先に観測項目を減らす
